常见问题

常见问题

可以在没有网络的情况下使用吗？

可以——安装后分析引擎完全离线运行。地图瓦片显示需要网络，但所有街区数据和评分计算都在本地设备上处理。搜索记录、收藏和自定义数据永远不会离开您的设备。

支持哪些城市和国家？

eodi.me 覆盖全球主要城市——来自 GeoNames 和 OpenStreetMap 的200多个国家15,000多个城市。覆盖范围在POI数据丰富的城市地区（欧洲、东亚、北美）最为完善。

街区数据如何采集？数据是最新的吗？

数据来自 OpenStreetMap（POI）、GeoNames（城市边界）和 Open-Meteo（气候）。数据集每季度更新一次。完全基于公开数据源构建。

支持 macOS 或 Linux 吗？

Windows 是主要支持平台。macOS 支持正在开发中。Linux 可通过商业计划的 Docker 部署使用。

许可证密钥如何工作？需要创建账户吗？

无需账户。通过 Polar 购买后，您将通过邮件收到许可证密钥。将其粘贴到应用的设置→许可证界面即可。一个密钥同时激活一台设备，取消激活后可用于其他设备。

取消订阅后我的数据会怎样？

收藏、预设和自定义位置都保存在本地设备上——从不上传到服务器。取消后14天宽限期过后降级为免费计划，本地数据保持不变。
